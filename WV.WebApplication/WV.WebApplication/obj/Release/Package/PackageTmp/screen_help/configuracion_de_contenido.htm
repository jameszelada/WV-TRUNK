2.3. Configuración de Contenido

 
 
.
Serie de Botones con los roles ingresados en el sistema, al presionar un rol en especifico éste desplegará en una tabla todo el contenido del sistema que tiene disponible.
 
 
Lista que permite elegir el numero de registros a ser mostrados en pantalla.
Botón que despliega una ventana emergente en la cual se listan todas las opciones disponibles en el sistema
 
Campo de texto que filtra registros que concuerden con el texto ingresado.
Botón que permite eliminar contenido del rol de sistema seleccionado.
 
Botón que despliega una ventana emergente permitiendo la elección de las acciones que se tendrán sobre el contenido del sistema elegido. Los permisos disponibles que serán desplegados en una opción son:
Agregar: Permite agregar registros dentro del contenido seleccionado.
Editar: Permite editar registros dentro del contenido seleccionado.
Eliminar:  Permite eliminar registros dentro del contenido seleccionado.
 
Cantidad de paginas en donde se visualizan los registros almacenados en el sistema.
 
 
Opción que despliega una pequeña ayuda sobre el funcionamiento de las opciones en la pantalla al habilitar el botón como "ON"
 
 
Opción que despliega el nombre del usuario con el que se esta logeado en el sistema, ademas muestra la opción Cerrar Sesion la cual termina la sesion del usuario actual y muestra el login para un nuevo ingreso.
