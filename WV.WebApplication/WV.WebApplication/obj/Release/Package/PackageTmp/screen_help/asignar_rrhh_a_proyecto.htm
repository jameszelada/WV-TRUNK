4.4. Asignar RRHH a Proyecto

 
Lista con todos los proyectos ingresados en el sistema. Ver Proyectos para mas información.
Botón que permite agregar un registro con la lista del Recurso Humano ingresado previamente al sistema, así como tambien una lista donde se puede elegir el puesto que desempeñará dentro del Proyecto seleccionado. El recurso humano solo podrá ser asignado a un proyecto que no tenga personal asignado previamente.
Botón que permite eliminar el registro de un Recurso Humano de la tabla de asignación a Proyectos, se eliminarán registros  mediante el check que se encuentra a la izquierda del nombre de la persona.
Check el cual permite seleccionar registros de personal asignado a Proyectos que se quieran eliminar.
Lista desplegable con todos los nombres del Recurso Humano registrado en el sistema
Lista desplegable con todos los puestos disponibles a ser asignados a una persona que se registre en un Proyecto que lleve a cabo la ONG.
Almacena los registros de Recursos Humanos diligenciados para un proyecto en especifico.
 
Mensaje de confirmación registro almacenado.
 
Cancela todos los cambios a realizase sobre un proyecto.
 
Mensaje confirmado si se cancela la operación
